# Portfolio — JOVALO.nl

Personal portfolio built with HTML, SCSS and vanilla JavaScript. Designed as a bento grid layout with support for dark and light mode.

## Tech

- HTML5 (semantic)
- SCSS (custom properties)
- Vanilla JavaScript

## Accessibility

Built with WCAG 2.1 AA in mind. Tested with:

- [WCAG.nl](https://wcag.nl)
- [WAVE Web Accessibility Evaluation Tool](https://wave.webaim.org)

Attention was paid to:

- Colour contrast in both dark and light mode
- Keyboard navigability and visible focus states
- Semantic HTML structure and heading hierarchy
- Respecting `prefers-reduced-motion` for animations
